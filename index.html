電腦基礎與硬體認識

計算機及電腦的發展史

電腦的定義在於「能執行預設指令序列」的機器，依照主要材料的不同， 主要分成四個不同的世代。
電腦的種類有超級電腦、大型主機、伺服器、 雲端運算、工作站、個人電腦 (PC)、智慧型手機及微電腦 (如 Arduino)。

三款具有里程碑意義的電腦：

Apple Macintosh (1984): 世界的重要一步。採用 Moto MC68000 處理器，具備 128 KB RAM，並內建 9 吋黑白螢幕與 3.5 吋軟碟機。

IBM PC 5150 (1981): 定義了現代 PC 標準。使用 Intel 8088 處理器，搭載 PC DOS 作業系統，開啟了 "Wintel" (Windows + Intel) 時代。

CER 710 / 小教授系列 (1980s): 台灣電腦工業的重要里程碑。 從早期的家用教育電腦演進至相容 IBM 標準的個人電腦。

現代電腦運作原理及種類

半導體與電晶體技術 (Semiconductor Manufacturing)

硬體縮小的關鍵在於電晶體技術的變革，從早期的點接觸電晶體、 雙極性接面電晶體 (BJT)，到現在主流的 MOSFET (金氧半場效電晶體) 與 FinFET (鰭式場效電晶體)，以及奈米技術的加持，電晶體結構變得 極其微小（如 22nm、10nm 甚至更小）。

邏輯閘與電路運算 (Logic Gates & TTL Signals)

電腦運算的基礎核心在於邏輯閘，基本邏輯閘像是 NOT (反閘) 、AND (及閘)、OR (或閘)、XOR (互斥或閘)、NAND 與 NOR， 將其彼此組合便能做出加法器 (Adder) 與 多工器 (Multiplexer)。 資料的儲存單位： 1 Byte = 8 bits；在電腦科學中，1 KB 等於 1024 Bytes。

電腦的訊號與數值系統 (Numeral Systems)

訊號處理有類比： 如 E6B 飛行計算尺、類神經網路訊號。 數位 (TTL 邏輯)就是使用電位區分 0 與 1， 0 (低電位)是0V ~ 0.5V；1 (高電位)是2.7V ~ 5V。

由於電腦訊號是以 0V/5V 代表 0 與 1，因此理解進位制轉換至關重要：
進位制轉換： 涵蓋二進位 (Binary)、八進位 (Octal)、十進位 (Decimal) 與十六進位 (Hexadecimal)。
計算邏輯： 透過「除法取餘數」將十進位轉為二進位； 透過「權位加總」將二進位轉回十進位。 例如：十六進位 $8A0 = 1000\ 1010\ 0000$ (二進位)。

硬體介紹

● 核心組件

中央處理器CPU

中央處理器與記憶體同步運作，包含算術邏輯單元 (ALU) 處理運算。 指令集架構主要分兩種，一種是RISC (精簡指令集)： 如 ARM (手機用)。 特點是指令簡單、執行速度快。而另一種是CISC (複雜指令集)： 如 Intel/AMD。特點是單一指令功能強大（如多媒體處理）。

記憶體(Memory)

記憶體分為兩種。DRAM (動態隨機存取記憶體)： 電腦主記憶體， 容量大、價格低，但速度較慢且需持續供電刷新。SRAM (靜態隨機存取記憶體)： 快取記憶體 (Cache)，速度極快、設計複雜、價格昂貴。

主機板 (Motherboard)

主機板的北橋 (Northbridge) 負責 CPU、記憶體與高速繪圖介面 (AGP/PCIe) 間的高速溝通。
而南橋 (Southbridge) 負責硬碟、USB、 音效等低速輸入輸出介面 (I/O)。

● 週邊設備與介面 (Peripherals & I/O)

儲存設備

HDD (硬碟)： 利用磁性讀取頭讀寫資料。
SSD (固態硬碟)： 使用 3D TLC NAND 快閃記憶體，讀取速度遠超傳統硬碟。

外部連接埠

顯示： VGA, DVI, HDMI, DisplayPort
傳輸： 序列埠 (RS232)、並列埠、USB。

網路

從 1960 年代的 300 bps 撥接數據機，發展到現代高達 400 Gbps 的乙太網路與無線 Wi-Fi。

程式設計與相關概念

什麼是程式設計？

程式設計是指編寫一段能被機器執行的指令序列，用來解決特定問題或 執行特定任務。而最原始的自動化程式就是批次檔 (.bat)， 在 DOS 或 Windows 中，將一系列指令寫在一個文字檔中，執行該檔案時， 系統會「依序」執行這些指令。

一個典型的程式通常包含以下邏輯結構：
順序結構 (Sequence)： 指令由上而下逐一執行。
選擇結構 (Selection/Condition)： 如 if...else，根據條件判斷要走哪一條路徑。
循環結構 (Loop)： 如 for, while，重複執行某段程式碼直到條件不成立。

程式語言依照用途可以分成系統程式與應用程式。系統程式 (System Programs) 顧名思義就是負責管理電腦硬體，提供平台給其他軟體運行。 例如：作業系統（Windows, Linux）、驅動程式、編譯器。
而應用程式 (Application Programs)則是為了滿足使用者特定需求而設計的程式。 例如：Word、瀏覽器、遊戲、計算機。

除了上述方法，我們也可以按照程式執行的主要方式分成 直譯型與編譯型語言 (Interpreted vs. Compiled Language) 編譯型 (Compiled) 程式碼在執行前，會先透過「編譯器」完整轉換成機器碼（.exe）。 優點是執行速度快、不需原始碼。例子： C, C++, Fortran。 直譯型 (Interpreted)程式碼執行時才由「直譯器」逐行將原始碼轉換為指令。 優點是跨平台方便、除錯容易。例子： Python, JavaScript, 早期的 BASIC。

程式語言的世代

第一代 (1GL)： 機器語言（0 與 1）。
第二代 (2GL)： 組合語言（Assembly）。
第三代 (3GL)： 高階語言（如 C, Java, Python），易於人類閱讀。
第四代 (4GL)： 接近自然語言，通常用於資料庫查詢（如 SQL）。
第五代 (5GL)： 用於人工智慧與神經網絡，強調解決問題的邏輯而非步驟。

必備應用程式

現代電腦通常需要安裝的基礎工具：
文書處理：Office 軟體、文字編輯器（如 Notepad++, PSPad）。
壓縮工具：WinRAR, 7-Zip。
網路工具：瀏覽器、FTP 客戶端。
開發工具：編譯器或整合開發環境 (IDE)。

HTML

HTML是什麼？

HTML(HyperText Markup Language)是一種標記文件結構與內容的解釋型標記語言，透過標籤組織資訊，並與CSS、JavaScript協作，形成完整的網頁運用。

HTML發展

  • 1991年Tim Berners-Lee發明
  • HTML 2.0 → HTML 4.01 → XHTML 1.0→ HTML5
  • 2014年HTML5成為正式標準

HTML5新功能

  • 語意化標籤：header、footer、section、article、aside、nav、hgroup
  • 圖形元素：canvas、svg
  • 多媒體元素:audio、video
  • API(Application Programming Interface，應用程式介面)擴充：地理定位(geolocation)、本地儲存(local storage)、拖放(drag and drop)、應用程式快取(application cache)、背景執行緒(web workers)、伺服器發送事件(server-sent events)
  • 移除標籤：acronym、applet、basefont、big、center、dir、font、frame、frameset、noframes、strike、tt

基礎架構

    <!DOCTYPE html>
    <html>
        <head>
            <meta charset="UTF-8">
            <title>網頁標題</title>
        </head>
        <body>
            網頁內容
        </body>
    </html>

head中的元素

CSS

CSS是什麼？

CSS(Cascading Style Sheets，階層式樣表)，它的工作是告訴瀏覽器：「這個標題要用什麼字體、多大、什麼顏色」、「那個段落和段落之間要留多少空間」、「這張圖片要靠左還是靠右」。

把CSS加到網頁的方法

1.行內樣式:直接在HTML標籤裡面用style屬性寫CSS
2.內部樣式表:在HTML的head裡面用style標籤寫CSS
3.外部樣式表:把CSS寫在一個獨立的檔案(例如 style.css)，再用link把它連結到HTML

基本語法

  選擇器 {
    屬性: 值;
    屬性: 值;
  }

CSS的選擇器

1.元素選擇器:直接寫標籤名稱，會選取頁面上所有該標籤
2.類別選擇器:寫一個點(.)，加上class名稱，會選取所有該class的元素
3.ID選擇器:寫一個#加上id名稱，會選取該id的元素(因為id是唯一的)
4.群組選擇器:用逗號分開，同時選取多種元素
5.後代選擇器:用空格分開，選取某元素「內部」的元素
6.偽類——針對元素的「狀態」:有些樣式要等到使用者做了某些動作才會套用

顏色表示法

1.顏色名稱：最直覺，但只有140種

 color: red;
 color: blue;
 color: lightcoral;
2.十六進位：最常用，以#開頭，六碼分別代表紅、綠、藍
 color:#FF0000;  /* 紅色 */
 color:#00FF00;  /* 綠色 */
 color:#0000FF;  /* 藍色 */
 color:#808080;  /* 灰色 */
3.RGB：用數字指定紅綠藍的含量，範圍0~255
 color: rgb(255, 0, 0);    /* 紅色 */
 color: rgb(0, 255, 0);    /* 綠色 */
 color: rgb(0, 0, 255);    /* 藍色 */
4.RGBA：RGB再加上透明度，透明度0是全透明，1是不透明
 color: rgba(255, 0, 0, 0.5);  /* 半透明的紅色 */

CSS的單位

1.絕對單位：大小是固定的
· px：像素，螢幕上的最小點
· pt：點，1pt=1/72英寸
· cm、mm：公分、公釐
2.相對單位：大小是相對於其他東西
· %：相對於父元素的百分比
· em：相對於目前元素的字體大小
· rem：相對於根元素(<html>)的字體大小
· vw：相對於視窗寬度的1%
· vh：相對於視窗高度的1%

CSS的盒子模型

在 CSS 的世界裡，每個元素都是一個盒子。這個盒子從內到外由四層組成：
1. 內容：實際的內容，像是文字或圖片
2. 內距：內容和邊框之間的距離
3. 邊框：盒子的邊框
4. 外距：盒子和旁邊其他元素之間的距離
在 CSS 裡，我們用這些屬性來控制：
1.width / height：內容的寬度與高度
2.padding：內距
3.border：邊框
4.margin：外距

元素的顯示方式

每個元素在網頁上都有預設的顯示方式：
1.區塊元素：像是<div>、<p>、<h1>，它們會佔滿整行，後面的元素會換到下一行。
2.行內元素：像是<span>、<a>、<strong>，它們會並排在同一行，不會強制換行。
3.行內區塊元素：像是 <img>、<button>，它們可以像行內元素一樣並排，但又可以像區塊元素一樣設定寬高。
我們可以用display屬性來改變元素的顯示方式：
1.display: block：變成區塊元素
2.display: inline：變成行內元素
3.display: none：隱藏元素(完全不佔空間)

文字的裝飾

1.字體：font-family設定要用什麼字體。瀏覽器會從左到右找，如果使用者電腦沒有第一個字體，就用第二個，依此類推。 

font-family: "微軟正黑體", Arial, sans-serif;
2.大小：font-size設定字體大小 
font-size: 16px;
font-size: 1.2rem;
3.粗細：font-weight設定字體粗細 
font-weight: normal;  /* 一般 */
font-weight: bold;    /* 粗體 */
4.對齊：text-align設定文字對齊方向 
text-align: left;    /* 靠左 */
text-align: center;  /* 置中 */
text-align: right;   /* 靠右 */
text-align: justify; /* 左右對齊 */
5.裝飾：text-decoration 加上底線或刪除線 
text-decoration: none;        /* 沒有裝飾，常用來移除連結底線 */
text-decoration: underline;   /* 底線 */
text-decoration: line-through;/* 刪除線 */
6.行高：line-height 設定行與行之間的距離 
line-height: 1.5;    /* 1.5 倍 */

背景的裝飾

1.背景顏色： 

background-color: lightblue;
2.背景圖片： 
background-image: url("圖片的路徑");
3.背景圖片如何擺放： 
background-repeat: no-repeat;   /* 不要重複 */
background-position: center;    /* 放在中間 */
background-size: cover;         /* 滿版，可能會裁切 */
background-size: contain;       /* 完整顯示，可能會留白 */

邊框與圓角

1.邊框需要同時設定粗細、樣式、顏色 

border: 1px solid black;      /* 1像素的黑色實線 */
border: 2px dashed red;       /* 2像素的紅色虛線 */
border: 5px dotted blue;      /* 5像素的藍色點線 */
2.圓角可以把方方正正的盒子變得柔和 
border-radius: 10px;          /* 四個角都圓 10 像素 */
border-radius: 50%;           /* 變成圓形（要搭配寬高相等） */

定位

1.相對定位：相對於元素原本的位置移動

position: relative;
top: 10px;
left: 20px;
/* 向下移動10像素，向右移動20像素 */
2.絕對定位：相對於最近的「有定位的祖先元素」來定位。如果都沒有，就相對於整個網頁
position: absolute;
top: 0;
right: 0;
/* 固定在右上角 */
3.固定定位：相對於瀏覽器視窗，不會隨著捲動而移動
position: fixed;
bottom: 20px;
right: 20px;
/* 固定在右下角，常用於「回到頂部」按鈕 */

JavaScript

JavaScript是什麼？

JavaScript是一種「程式語言」，跟HTML和CSS完全不同。HTML只能標記內容，CSS只能設定樣式，但JavaScript可以：

  • 回應使用者的操作
  • 計算數學
  • 動態修改網頁內容
  • 跟伺服器交換資料
  • 控制多媒體播放
JavaScript能使網頁成為可以互動的應用程式。

JavaScript的歷史

  • JavaScript是1995年由Netscape公司的Brendan Eich設計出來的。
  • 原本JavaScript只能跑在瀏覽器裡，但現在透過Node.js，JavaScript也可以寫伺服器程式，可以說是目前全端開發最熱門的語言

把JavaScript加到網頁

1.行內寫法： 

<button onclick="alert('按鈕被按了')">點我</button>
2.內部寫法： 
<script>
     function sayHello() {
        alert('Hello!');
    }
</script>
3.外部寫法：

變數

寫程式常常需要暫存一些資料，這時候就需要「變數」。
1.let宣告的變數可以改變內容
2.const宣告的變數是常數，一旦設定就不能改變

資料型別

1.數字：包含整數和小數
2.字串：就是一串文字，要用引號包起來
3.布林值：只有true(真)和false(假)
4.undefined：變數有宣告但沒給值的時候
5.null：空值，通常是我們手動設定的

運算子

1.算術運算子：

  • +：加法
  • -：減法
  • *：乘法
  • /：除法
  • %：取餘數
2.比較運算子：
  • >：大於
  • <：小於
  • >=：大於等於
  • <=：小於等於
  • ===：相等(嚴格比對，連型別都要一樣)
  • !==：不相等
3.邏輯運算子：
  • &&：且(兩個條件都要成立)
  • ||：或(只要一個條件成立)
  • !：反過來(true變false，false變true)

條件判斷——如果怎樣就怎樣

程式常需要根據不同情況做不同的事，這時就需用if：
ex:

let 分數 = 85;

if (分數 >= 90) {
    console.log("拿到 A");
} else if (分數 >= 80) {
    console.log("拿到 B");
} else {
    console.log("拿到 C");
}
意思是：
· 如果分數大於等於 90，顯示「拿到 A」
· 不然如果分數大於等於 80，顯示「拿到 B」
· 以上都不是，顯示「拿到 C」

迴圈——重複做某件事

有時需要重複做同一件事很多次，這時就用迴圈。
for 迴圈：適合「知道要跑幾次」的情況 

for (let i = 1; i <= 5; i++) {
    console.log("這是第 "+i+" 次");
}
// 會印出 1 到 5
while 迴圈：適合「不知道要跑幾次，但知道什麼時候停止」的情況 
let 次數 = 1;
while (次數 <= 5) {
    console.log("這是第 " + 次數 + " 次");
    次數++;
}

函式

當我們有一段程式碼會重複用到，就可以把它包成「函式」: 

function 打招呼(名字) {
    console.log("哈囉，我是" + 名字);
}

// 使用函式
打招呼("小明");  // 印出「哈囉，我是小明」
打招呼("小華");  // 印出「哈囉，我是小華」
函式也可以「回傳」計算結果： 
function 加法(a, b) {
    return a + b;
}

let 結果 = 加法(5, 3);  // 結果等於 8

陣列

陣列很常用，可以用來處理一連串的資料： 

let 水果 = ["蘋果", "香蕉", "橘子"];
console.log(水果[0]);  // 顯示「蘋果」（索引從 0 開始）
console.log(水果[1]);  // 顯示「香蕉」

物件

物件用「名稱:值」的組合來儲存資料： 

let 學生 = {
    姓名: "小明",
    年齡: 18,
    分數: 85,
    打招呼: function() {
        console.log("大家好，我是" + this.姓名);
    }
};

console.log(學生.姓名);  // 顯示「小明」
學生.打招呼();           // 顯示「大家好，我是小明」

DOM——JavaScript和HTML的橋樑

DOM(Document Object Model，文件物件模型)，它可以把HTML文件轉換成JavaScript可以操作的結構。
透過 DOM，JavaScript 可以：
· 找到某個 HTML 元素
// 用 id 找
let 標題 = document.getElementById("main-title");

// 用 class 找（會找到多個）
let 按鈕們 = document.getElementsByClassName("btn");

// 用 CSS 選擇器找（較現代的方式）
let 第一個段落 = document.querySelector("p");
let 所有段落 = document.querySelectorAll("p");
· 修改那個元素的內容
let 標題 = document.getElementById("main-title");
標題.textContent = "新標題";          // 純文字
標題.innerHTML = "<span>新標題</span>"; // 可以放 HTML 標籤
· 修改那個元素的樣式
let 盒子 = document.getElementById("box");
盒子.style.color = "red";
盒子.style.backgroundColor = "black";
盒子.style.fontSize = "20px";
· 監聽那個元素的事件

事件

「事件」就是使用者對網頁做的動作，例如：
· 點擊按鈕
· 移動滑鼠
· 在輸入框打字
· 捲動頁面
我們可以用 JavaScript 來「監聽」這些事件，並做出回應：
ex:

let 按鈕 = document.getElementById("myButton");
按鈕.addEventListener("click", function() {
    alert("你按了按鈕！");
});
這段的意思是：找到 id 是 "myButton" 的元素，然後監聽它的「點擊」事件，當事件發生時，執行後面的函式。
